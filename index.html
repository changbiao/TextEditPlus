TextEditPlus

An enhanced version of TextEdit in Mac OS X, features encoding detection, etc.

View project onGitHub

In addition to the original TextEdit from Apple, TextEditPlus only does two things:

  • Automatically detects text encoding when opening a file. (When you choose to do so.)
  • Make sure each line has the same height when there are mixed scripts in the same lines using fallback fonts.

Encoding detection is using the awesome UniversalDetector framework
